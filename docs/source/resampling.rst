==========
Resampling
==========

.. py:module:: eniric.resampling
    :synopsis: Spectral resampling.

In Figueria et al. (2016) and :mod:`scripts.phoenix_precision` the synthetic spectra are resampled to
3 pixels per resolution element.

This is done using the :mod:`eniric.resample` module.
